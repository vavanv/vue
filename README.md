# Vue Next Gen - Employee Directory

A modern, high-performance Vue 3 application built with Vite and PrimeVue 4, featuring a sophisticated design system and advanced data grid capabilities.

## âœ¨ Features

- **Vue 3 + Vite**: Leveraging the fastest development environment and the latest Vue composition API.
- **PrimeVue 4 Integration**: Fully integrated with the **Aura** theme for professional, accessible UI components.
- **Sophisticated Design System**:
  - **Glassmorphism**: Modern frosted-glass aesthetic throughout the interface.
  - **Theme Support**: Seamless Dark/Light mode toggle with persistence.
  - **HSL Color Palette**: A refined, modern color system.
- **Advanced Employee Directory**:
  - **High-Performance DataTable**: Efficiently handles 200+ mockup records.
  - **Universal Height Stability**: Custom flex-scroll architecture ensures the grid maintains a constant height across all page sizes (10, 20, 50, 100).
  - **Sticky Headers**: Essential for data clarity during scrolling.
  - **Zero-Scrollbar Environment**: Smart height calculation prevents global browser scrollbars, maintaining an "app-like" feel.
  - **Dynamic Responsiveness**: Auto-calculates available space on window resize while maintaining a comfortable bottom buffer.
